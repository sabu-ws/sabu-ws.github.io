Wath is SABU ?

SABU, which stands for Secure Against Bad USB Key, is an innovative project carried out by three students as part of cybersecurity training. SABU presents itself as a state-of-the-art whitestation based on a client-server model. It is equipped with the latest generation detection means, capable of identifying and reporting viruses present on your employees' USB keys. This project aims to provide a robust solution for securing USB devices, ensuring that harmful malware does not compromise your systems.

Whitestation

The whitestation concept is based on the idea of ​​providing a secure and controlled environment for the use and processing of USB devices. A whitestation serves as a buffer zone where USB devices can be examined and cleaned before being connected to the company's main network.

Features

Simplified Configuration

The SABU whitestation is equipped with an optimized administration panel that makes it easy to configure and manage all options. Thanks to this user-friendly interface, you can administer the server as well as each endpoint quickly and easily.

Centralized Endpoint Management

SABU facilitates centralized endpoint management. You can monitor and control each endpoint from one place, ensuring efficient and hassle-free administration.

File Storage System

SABU has an integrated storage system allowing your employees to save files from their USB keys after scanning them. This ensures that only secure files are stored and used in your work environment.

Protocol Break

Thanks to its client-server architecture, SABU ensures an efficient protocol break. Files can flow through the SABU system securely, minimizing the risk of malware spreading. This allows your employees to transfer their files in a secure and controlled manner.

User Management

SABU includes advanced user management features, allowing you to define specific roles and permissions for each employee. This ensures that each user only accesses functionality and data relevant to their role.

Advanced Virus Detection

The SABU whitestation uses latest generation virus detection technologies, capable of identifying and neutralizing the most recent and sophisticated threats. This advanced protection is essential for preventing malware attacks via USB devices.

Frequently Asked Questions

How is SABU performing ?

SABU is light enough to operate efficiently. However, the performance of your server will directly influence the time required for scans. A more powerful server will allow faster scans.

How many endpoints can I manage with SABU ?

There is no configuration limit on the number of endpoints SABU can manage. The only restriction depends on the capacity of your server to support the scan load.

SABU is free ?

Yes, SABU is completely free and open-source. The community is encouraged to consult and improve the code, thus contributing to the evolution and improvement of the project.
